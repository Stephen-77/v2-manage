# vue-manage
项目概述: 基础的后台管理系统，实现登录，个人信息，权限管理

技术选型：vue2 + vuex + elementUI+ echarts + mock
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
